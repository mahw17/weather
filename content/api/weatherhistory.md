Weather history - API
===========================================


##API syntax

Baserat p책 en IP-adress eller p책 koordinater kan du h채mta 30dagars summerad v채derhistorik och returnera svaret i en JSON-struktur

```text
GET /weather/history/<end date>/<source>/<ip-adress || lat/lon>
```

Till exempel
med ip...
```text
GET /weather/history/1542829827/ip/208.80.152.201
```
[Testa IP-adress '208.80.152.201'](weather/history/1542829827/ip/208.80.152.201)

eller med koordinater...
```text
GET /weather/history/1542829827/coord/55.61/13
```
[Testa koordinater lat=55.6 och longitud=13](weather/history/1542829827/coord/55.61/13)
