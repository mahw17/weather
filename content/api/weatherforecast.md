Weather forecast - API
===========================================


##API syntax

Baserat p책 en IP-adress eller p책 koordinater kan du h채mta en v채derprognos och returnera svaret i en JSON-struktur

```text
GET /weather/forecast/<source>/<ip-adress || lat/lon>
```

Till exempel
med ip...
```text
GET /weather/forecast/ip/208.80.152.201
```
[Testa IP-adress '208.80.152.201'](weather/forecast/ip/208.80.152.201)

eller med koordinater...
```text
GET /weather/forecast/coord/55.61/13
```
[Testa koordinater lat=55.6 och longitud=13](weather/forecast/coord/55.61/13)
